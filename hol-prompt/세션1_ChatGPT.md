**One Agenda, Four AI · kt estate 임원 AI 활용 실습교육 · 프롬프트북 v2**

> **사용법**
> - 각 코드 블록 우측 상단의 복사 버튼으로 프롬프트를 복사해 ChatGPT에 붙여넣습니다.

## STEP 1. 워밍업 (Mobile Phone ChatGPT Live Voice)

날씨 -> 기분 (우울, 기쁨) -> 상황설명 -> 솔루션 도출

## STEP 2. 프로젝트 생성

아래 순서대로 진행해 주세요.

1. ChatGPT 왼쪽 사이드바에서 **프로젝트 → 새 프로젝트**를 선택합니다.
2. 프로젝트 이름을 **`프로젝트 리본`**으로 입력하고 생성합니다.

> 프로젝트는 대화·파일·지침을 하나의 업무 공간으로 묶어 줍니다. 이제부터 리본 관련 대화는 모두 이 프로젝트 안에서 진행합니다.

---

## STEP 3. 프로젝트 지침 개인화

프로젝트 화면에서 **프로젝트 설정 → 지침(Instructions)** 을 열고, 아래 내용을 복사해 붙여넣은 뒤 저장해 주세요.

### 프로젝트 지침 (맥락 + 역할 + 답변 규칙)
```markdown
# 프로젝트 맥락
- 회사: kt estate (부동산 개발·운영 전문기업)
- 안건: 서울 ○○동 구 KT지사 유휴부지(연면적 약 5,000평) 복합개발 전략 수립
- 프로젝트 코드명: 리본(Re:Born) — 구 통신 인프라 부지의 재탄생
- 개발 목표: 임대수익 안정성과 자산가치 상승 동시 확보

# 나의 역할
- 나는 kt estate Re:Born TF 내 [OO담당] 임원이다.
- 모든 답변은 임원의 의사결정을 돕는 관점으로 작성하라.

# 답변 규칙
- 한국어로, 결론부터 제시하라.
- 표와 개조식 요약을 우선 사용하라.
- 수치와 사실에는 근거를 함께 표시하고, 불확실한 내용은 '확인 필요'로 표시하라.
```

> **티칭 포인트:** 매 대화마다 맥락 카드를 붙여넣는 대신, 프로젝트 지침에 한 번 넣어두면 이 프로젝트 안의 **모든 새 대화에 자동 적용**됩니다.

## STEP 4. 지침 적용 확인

프로젝트 안에서 **새 대화**를 열고, 아래 내용을 복사해 주세요.

### 개인화 확인 질문
```markdown
- 이 프로젝트의 맥락과 나의 역할, 네가 지켜야 할 답변 규칙을 세 줄로 요약해줘
```

> 아무 설명도 하지 않았는데 AI가 프로젝트와 나를 알고 있다면 — 지침 개인화 성공

---

## STEP 5. Deep Research 시장 심층 조사

프로젝트 안 새 대화에서 **Deep Research 모드**를 선택하고, 아래 내용을 복사해 주세요.

### 시니어 주거 시장 조사
```markdown
# 조사 주제
- 국내 시니어 주거·레지던스 시장 심층 조사

# 조사 배경
- 구 KT지사 유휴부지(서울 도심, 연면적 약 5,000평) 복합개발 컨셉 검토용

# 포함할 내용
- 최근 2년 공급 사례와 운영사
- 수요 전망 (인구·소득 구조 변화 포함)
- 관련 정책·규제 동향
- 해외 선진 사례 2~3건

# 출력 조건
- 모든 수치와 사실에 출처를 명시하라.
- 마지막에 '유휴부지 복합개발 시사점'을 5줄로 요약하라.
```

> Deep Research가 도는 동안 기다리지 않습니다 — 바로 STEP 6으로 진행합니다.

## STEP 6. 개발 컨셉 브레인스토밍 (기본형)

아래 내용을 복사해 주세요. 맥락은 프로젝트 지침이 이미 품고 있으므로, **역할과 형식만으로** 프롬프트가 완성됩니다.

### 개발 컨셉 5가지
```markdown
# 역할
- 너는 20년 경력의 부동산 개발 전략 컨설턴트다.

# 요청
- 이 프로젝트 맥락에 맞는 개발 컨셉 5가지를 제안하라.

# 출력 형식
- 표로 정리하라.
- 표 항목: 컨셉명, 핵심 타깃, 기대 효과, 주요 리스크
```
