## STEP 2. 소스 업로드 후 아래 프롬프트 입력

- 자료에 없는 내용은 추측하지 말고 '자료에 근거 없음'으로 표시하라.
- 2000자 이내로 요약해라

## STEP 3. 스튜디오 오버뷰 생성 (선실행)

프롬프트 없이 기능으로 실행합니다. 아래 순서대로 진행해 주세요.

1. **오디오 오버뷰(한국어) 생성**을 클릭합니다 — 두 AI 진행자가 '프로젝트 리본'을 대담하는 팟캐스트가 만들어집니다. ★와우 모먼트
2. 생성에 수 분이 걸리므로 걸어둔 채 STEP 4로 진행합니다. 완료되면 하이라이트 1–2분만 함께 청취합니다.

## STEP 4. 보고 슬라이드 생성

아래 내용을 복사해 주세요.

### 임원 보고용 슬라이드
```markdown
# 요청
- 임원 보고용 슬라이드를 만들어라.
- 제목: '프로젝트 리본 — 시장 동향과 개발 시사점'
- 리스크 최소화 관점으로 작성

# 구성 가이드
- 표지 1장, 시장 동향 2장, 시사점 1장, 다음 단계 1장
```

## STEP 5. 슬라이드 다운로드 — 그리고 반전

아래 순서대로 진행해 주세요.

1. 생성된 슬라이드를 **다운로드(PDF)** 합니다. 파일명 예: `리본_시장동향_슬라이드.pdf`
2. PowerPoint(또는 뷰어)에서 열어 글자를 클릭해 봅니다 → **수정되지 않습니다.** (이미지 기반의 한계 체감) ★반전 연출

> **한계 해결의 열쇠:** "AI의 한계는 다른 AI로 잇고, 잇는 언어는 마크다운이다." 편집 가능한 원문(개요)을 추출해 다음 세션으로 넘깁니다.
> 이 PDF는 **세션 4(Copilot) 실습 파일**로도 사용됩니다 — 보관해 주세요.

## STEP 6. 마크다운 개요 추출 — 릴레이의 바통 ★핵심

아래 내용을 복사해 주세요.

### 장표별 텍스트 개요
```markdown
# 요청
- 방금 만든 슬라이드 내용을 장표별 텍스트 개요로 정리하라.

# 출력 형식
- 각 장표의 제목은 '#'으로 시작하는 줄로 써라.
- 각 장표의 본문 요점은 '-'로 시작하는 줄로 써라.
- 서식 설명 없이 개요 텍스트만 출력하라.
```

> **이 출력물(마크다운 개요)을 복사해 둡니다 — 세션 3(Claude Design 슬라이드 생성)에서 재사용됩니다.**
